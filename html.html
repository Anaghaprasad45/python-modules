html \xe2\x80\x94 HyperText Markup Language support\xc2\xb6

\n

Source code: Lib/html/__init__.py

\n
\n

This module defines utilities to manipulate HTML.

\n
\n
\nhtml.escape(s, quote=True)\xc2\xb6
\n

Convert the characters &, < and > in string s to HTML-safe\nsequences. Use this if you need to display text that might contain such\ncharacters in HTML. If the optional flag quote is true, the characters\n(") and (\') are also translated; this helps for inclusion in an HTML\nattribute value delimited by quotes, as in <a href="...">.

\n
\n

New in version 3.2.

\n
\n
\n\n
\n
\nhtml.unescape(s)\xc2\xb6
\n

Convert all named and numeric character references (e.g. &gt;,\n&#62;, &#x3e;) in the string s to the corresponding Unicode\ncharacters. This function uses the rules defined by the HTML 5 standard\nfor both valid and invalid character references, and the list of\nHTML 5 named character references.

\n
\n

New in version 3.4.

\n
\n
\n\n
\n
